# Thư mục hình ảnh báo cáo VietForces

Đặt các file ảnh chụp màn hình (screenshot) của app vào đây.

## Tên file được tham chiếu trong `main.tex`

| Tên file | Nội dung |
|---|---|
| `screenshot_home.png` | Màn hình chính (Home) |
| `screenshot_game_image_to_word.png` | Game mode: Nhìn hình đoán từ |
| `screenshot_game_word_to_image.png` | Game mode: Nghe từ chọn hình |
| `screenshot_game_fill_blank.png` | Game mode: Điền từ còn thiếu |
| `screenshot_game_sentence_order.png` | Game mode: Sắp xếp câu |
| `screenshot_game_word_chain.png` | Game mode: Chuỗi từ |
| `screenshot_game_syllable_match.png` | Game mode: Ghép âm tiết |
| `screenshot_game_word_search.png` | Game mode: Tìm từ trong bảng chữ |
| `screenshot_leaderboard.png` | Màn hình bảng xếp hạng (real-time) |
| `screenshot_daily_challenge.png` | Màn hình Daily Challenge |
| `screenshot_profile.png` | Màn hình hồ sơ cá nhân (streak calendar) |
| `screenshot_social.png` | Màn hình bạn bè / activity feed |
| `screenshot_ai_roleplay.png` | AI Roleplay conversation |
| `screenshot_ai_writing.png` | AI Writing Practice |
| `screenshot_login.png` | Màn hình đăng nhập |
| `screenshot_admin.png` | Web Admin Dashboard |
| `screenshot_landing.png` | Web Landing Page |

## Định dạng khuyến nghị
- PNG hoặc JPG
- Độ phân giải ≥ 1080 × 1920 px (portrait) cho screenshot điện thoại
- Screenshot Admin/Landing có thể landscape 1920 × 1080 px
